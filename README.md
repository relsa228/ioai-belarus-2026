# IOAI-2026 Belarus
Tasks for national stage of the IOAI-2026. Held from March 31 to April 2, 2026 in Minsk, Republic of Belarus.

## Task 1: Galactic Morphological Classification
Analysis of multi-channel (RGB) astronomical imagery to identify and categorize galaxy structures. The model must distinguish between various morphological types, such as spiral, elliptical, and irregular galaxies, based on their visual features.

## Task 2: Martian Surface Topography Analysis
Processing of panchromatic (grayscale) orbital imagery to identify distinct geological features on the Martian surface. The task involves recognizing surface patterns, crater types, and terrain textures essential for landing site selection and rover navigation.

## Task 3: Transient Meteor Detection (Differential Imaging)
High-precision detection of meteoric events using image subtraction (comparison). The system must analyze temporal changes between consecutive frames to distinguish fast-moving celestial objects from static background noise and sensor artifacts.
